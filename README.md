### `//` Featured Labs

#### 🔐 Cybersecurity Home Lab — `VirtualBox` / `Kali Linux`
Isolated environment for practicing defense-in-depth, vulnerability management, and SIEM-based detection.
- Identified **20+ open ports** via Nmap; reduced attack surface to **2** through firewall hardening
- Analyzed traffic with **Wireshark**, flagging port scans and insecure protocols in real time
- Ran authenticated vulnerability scans with **OpenVAS** and prioritized remediation by severity
- Applied **CIS benchmarks** — raised system security score from **42 → 74**
- Deployed **Wazuh SIEM** with custom alerts for brute-force attempts and recon activity

#### 🌐 Enterprise Network Build — `Cisco Packet Tracer`
Segmented multi-department network modeled on real enterprise topology.
- Configured **VLANs**, **trunking**, **inter-VLAN routing**, and **OSPF**
- Deployed **DHCP** and **DNS** for automated network management
- Enforced **ACLs**, **port security**, and **wireless segmentation** as layered controls
